Política de Privacidad

Última actualización: Octubre 2025

En Reservas UNAB respetamos la privacidad de nuestros usuarios. A continuación describimos qué datos recopilamos, cómo los usamos y qué opciones tienes como usuario.

1. Información que recopilamos

Podemos recopilar y procesar la siguiente información:

  • Información de cuenta o identificador (por ejemplo, nombre de usuario o correo) proporcionada al registrarse.
  • Archivos que el usuario sube voluntariamente (por ejemplo, imágenes para reportes o perfil).
  • Datos técnicos de conexión y uso (por ejemplo, registros de errores, direcciones IP y tokens de sesión) para operar y mejorar el servicio.

2. Cómo usamos la información

  • Para autenticar y autorizar el acceso a tu cuenta.
  • Para permitir la comunicación con nuestro backend y procesar funcionalidades (consultas, generación de reportes, descargas).
  • Para procesar pagos mediante pasarelas externas que el usuario utiliza para realizar transacciones.
  • Para diagnosticar y corregir fallos, y mejorar la experiencia de la aplicación.

3. Compartición de la información

No vendemos ni alquilamos tu información a terceros. Solo compartimos datos cuando es necesario para prestar el servicio:

  • Con la pasarela de pagos que utilizas para procesar transacciones (ej. Ecollect o similar), únicamente la información necesaria para realizar el cobro.
  • Con proveedores de infraestructura (hosting, almacenamiento) que procesan datos bajo nuestras instrucciones.
  • Si la ley nos obliga o para proteger derechos legales.

4. Permisos y acceso al dispositivo

La aplicación solicita únicamente los permisos necesarios para su funcionamiento. Entre ellos:

  • INTERNET: para conectar la app con nuestro backend y con la pasarela de pagos.
  • READ_EXTERNAL_STORAGE / READ_MEDIA_IMAGES: para que los usuarios puedan seleccionar y subir imágenes desde su dispositivo.

No se accede al micrófono, a la cámara ni a la ubicación del usuario, salvo que el usuario lo autorice explícitamente en futuras versiones.

5. Seguridad

Implementamos medidas técnicas y administrativas razonables para proteger la información contra pérdida, acceso no autorizado o divulgación. Sin embargo, ningún método de transmisión por Internet o almacenamiento electrónico es 100% seguro.

6. Conservación de datos

Mantenemos los datos mientras sea necesario para los fines descritos (operación de la app, cumplimiento legal o para mejorar el servicio). Si quieres que eliminemos tu cuenta y datos, contáctanos (ver abajo).

7. Tus derechos

Puedes solicitar acceder, corregir o eliminar tus datos, o presentar una queja sobre el tratamiento de los mismos. Para ello contáctanos en el correo indicado más abajo.
